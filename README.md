## Getting Started
Most of the projects would run with npm commands.


### Prerequisites
NPM, Node.js

### Installing
This would install all the dependencies
```
$ npm install
```

### Running Locally
```
$ npm start
```
This app runs on [localhost:3000](http://localhost:3000/).
